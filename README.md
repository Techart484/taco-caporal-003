# Taco Caporal

Taco Caporal offers a vibrant taste of Mexico through authentic, fresh, and flavorful tacos and traditional dishes, providing a convenient and satisfying dining experience.

## Getting Started

This is a modern Next.js 15 application built with:
- Next.js 15 with App Router
- TypeScript
- Tailwind CSS 4
- shadcn/ui components

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the result.

## Build

```bash
npm run build
npm start
```

## Tech Stack

- **Framework**: Next.js 15
- **Language**: TypeScript
